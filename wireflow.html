CareConnect · Family Portal · v0.0

Find them the right next home.

Sixteen mobile screens for adult children placing a parent or loved one in senior care. The product's core insight: families are stressed and information-starved — brochures and Google reviews don't tell them what they need to know. Four mechanisms make this app different from a directory: the data model treats the user (the family member) and the loved one (the prospective resident) as separate but linked records — and that split is honored everywhere, including in two distinct edit screens (Sarah's account vs Elizabeth's profile) so the two records never co-edit in one form; OpenDoor Live gives families recorded, unscripted Q&A access to home directors; Hidden Gems surfaces small, well-run operators that don't rank on Google; and chat is scoped per-intro inside the Active Intros pipeline, never a global inbox — so every conversation belongs to a specific lead with its own status timeline.

User flow

Primary paths through the app

Five paths cover every screen. Bottom-tab navigation between Search, Active Intros, and Profile is always available from those three screens — the rest are entered through specific actions, all of which are labeled below each phone in the canvas. Conversations with homes live inside an intro (Path 5), not in a global inbox tab — every message belongs to a specific lead. Intake and loved-one editing also capture a cultural/religious-fit preference (including events heavily celebrated by each home), and that signal feeds match scoring in Search and Home Profile.

Screen 16 Edit Elizabeth's profile Dedicated edit screen for the loved-one record, reached from the "Edit" link on the "Caring for" card on Profile (Screen 14).
